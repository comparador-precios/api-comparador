## Clonar proyecto

```bash
git clone https://github.com/comparador-precios/api-comparador.git
```

## Instalar dependencias

```bash
npm install
```

## Ejecutar migracion de Prisma

```bash
npx prisma migrate dev --name init
```

## Levantar el servidor

```bash
# development
npm run start

# watch mode
npm run start:dev

# production mode
npm run start:prod
```
